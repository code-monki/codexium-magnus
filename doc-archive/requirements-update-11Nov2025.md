<h3 id="fr-11">[FR-11] Typography</h3>

**Description:**  
The system shall support consistent, user-adjustable typography across all viewer-rendered content.

| Requirement | Description |
|------------|-------------|
| <a id="fr-11-1" />[FR-11.1] | Provide a default typography preset (base font, headings, spacing) defined by the cartridge. |
| <a id="fr-11-2" />[FR-11.2] | Allow users to override typography: base font size (relative scaling), line height, and paragraph spacing. |
| <a id="fr-11-3" />[FR-11.3] | Support at least one bundled sans-serif font ("Inter" or "Roboto") and allow cartridges to supply additional fonts. |
| <a id="fr-11-4" />[FR-11.4] | Ensure WCAG-compliant contrast and readability for both light and dark themes. |
| <a id="fr-11-5" />[FR-11.5] | Store typography preferences per user, not per cartridge. |

<h3 id="fr-12">[FR-12] Bibliography</h3>

**Description:**  
The system shall support generation and rendering of bibliographies for articles and collections.

| Requirement | Description |
|------------|-------------|
| <a id="fr-12-1" />[FR-12.1] | Support APA and CMS bibliography formatting rules. |
| <a id="fr-12-2" />[FR-12.2] | Bibliography entries shall be generated programmatically from metadata stored in the cartridge. |
| <a id="fr-12-3" />[FR-12.3] | On-screen and printed bibliographies must follow the same ordering and formatting rules, unless overridden by print layout. |
| <a id="fr-12-4" />[FR-12.4] | Support grouped bibliographies for multi-article print jobs (e.g., print queue or booklet export). |
| <a id="fr-12-5" />[FR-12.5] | Allow cartridges to embed custom citation metadata (e.g., URL, DOI, ISSN, publisher) without modifying the viewer. |
